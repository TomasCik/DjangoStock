Projektas - DjangoStock


Projekto aprašymas:



DjangoStock yra mokomasis projektas, sukurtas naudojant Python Django karkasą. Jis leidžia vartotojams stebėti akcijų kainas, pridėti jas į portfolio ir gauti detalią peržiūrą apie įvairias akcijas.

Naudojamos Technologijos:

  • Django: Pagrindinis svetainės karkasas.
  • yfinance: Biblioteka, skirta gauti akcijų duomenis.
  • Seaborn ir Matplotlib: Grafikų generavimui.
  • requests: API užklausoms.
  • IEX Cloud: Akcijų informacijos API.
  • Bootstrap: Tinklapio dizaino elementai.
